Google OAuth Client ID
Need a Client ID? It's free and takes ~5 minutes. Show me how
1. Go to console.cloud.google.com and create a free project
2. Go to APIs & Services → OAuth consent screen — choose "External", add your email as a test user
3. Go to APIs & Services → Credentials → Create Credentials → OAuth client ID
4. Application type: Web application
5. Under "Authorized JavaScript origins" add: null and the URL you host this file at (if any)
6. Go to APIs & Services → Library, search "Google Drive API" and enable it
7. Copy the Client ID and paste it above
